# Loaded
**Loaded** is a network-distributed build agent that can perform arbitrary tasks on different computers.

Check out the [documentation](http://loaded.readthedocs.org/en/latest/?badge=latest) to find out more !

## Installation

Just type:

> pip install loaded

That's it.
